# **App Name**: Ripple: Video Calling

## Core Features:

- Start Call: Initiate video calls with contacts.
- Join Session: Join active video conference sessions using a session ID.
- Real-Time Chat: Real-time text chat during video calls.
- Contact Management: Manage and organize a list of contacts.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to evoke a sense of trust and reliability, integral for communication applications. This color is balanced and inviting.
- Background color: Off-white (#F9F9F9), almost the same hue as the primary, but highly desaturated (approx. 10%). It provides a clean, modern look.
- Accent color: A subdued teal (#70CBC0) that harmonizes with the blue and enhances visual elements without being distracting.
- Body and headline font: 'Inter', a sans-serif font known for its modern, objective, and neutral appearance, ensures excellent readability across different screen sizes and resolutions.
- Clean, minimalist icons using the primary blue color to maintain consistency and clarity.
- Clean and intuitive layout with easy navigation for quick access to main features.
- Engaging and cool transitions and animations to enhance user experience and provide feedback on interactions. Animations should be modern and subtly draw the user's attention.
- UI elements such as buttons and panels should have a glass-like appearance.
